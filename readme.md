# 六面骰模拟器

**v 1.0.0r | Github开源版**

六面骰模拟器是一个模拟六面骰功能的程序。

使用六面骰模拟器可以模拟投掷六面骰并获得掷出的点数，六面骰模拟器还支持导入不同的六面骰以及修改六面骰的参数。

此版本六面骰模拟器开发完成日期：2026年6月28日  
开发者：BookDarksteel

## 如何开始使用六面骰模拟器

只需将Sixsided-Dice-Simulator项目的所有文件保存到一个文件夹中，然后运行其中的 *mian.py* 。

## 关于运行环境

六面骰模拟器的图形交互界面基于Tkinter开发。  
六面骰模拟器的图形交互界面中使用“思源宋体”字体，您可能需要先自行在计算机上安装“思源宋体”字体，才能在使用六面骰模拟器时体验到比较理想的效果。

六面骰模拟器使用了如下Python库，这些库都是比较常见的。

- atexit
- copy
- os
- random
- shutil
- sys
- tkinter

六面骰模拟器的开发与测试均在Windows 11操作系统下进行，使用的Python解释器版本为3.10。

## 关于六面骰模拟器中的六面骰

六面骰模拟器的项目文件中包含两个六面骰的信息文件，这些信息文件已经被导入六面骰模拟器的对应工作文件夹，您只需正常启动六面骰模拟器就可以使用它们。

# Sixsided Dice Simulator

**v 1.0.0r | Github open source edition**

Sixsided Dice Simulator is an application that simulates the functionality of sixsided dice.

By using the Sixsided Dice Simulator, you can simulate rolling sixsided dice and obtain the results. The Sixsided Dice Simulator also supports importing different sixsided dice and modifying their parameters.

Development completion date for this version of Sixsided Dice Simulator: June 28, 2026  
Developer: BookDarksteel

## How to get started with Sixsided Dice Simulator

Simply save all the files of the Sixsided-Dice-Simulator project to a folder and run *mian.py* from it.

## About the running environment

The graphical user interface of the Sixsided Dice Simulator was developed using Tkinter.  
The graphical user interface of the Sixsided Dice Simulator uses the “Source Han Serif CN” font. You may need to install the “Source Han Serif CN” font on your computer first to ensure optimal performance when using the Sixsided Dice Simulator.

Sixsided Dice Simulator uses the following Python libraries, which are relatively common.

- atexit
- copy
- os
- random
- shutil
- sys
- tkinter

Sixsided Dice Simulator was developed and tested under the Windows 11 operating system, using Python interpreter version 3.10.

Apologies that due to the lack of knowledge of the developers of Sixsided Dice Simulator and the limited testing conditions, it is unable to give more information about compatibility here.

## About the interactive interface

Sixsided Dice Simulator has graphical user interface based on Tkinter.

Although this *readme.md* file contains descriptions of the Sixsided Dice Simulator in both Chinese and English, the text in the Sixsided Dice Simulator’s graphical user interface is primarily in Chinese. You may need to be able to read simple Chinese in order to use the Sixsided Dice Simulator easily.

## About the sixsided dice in the Sixsided Dice Simulator

The Sixsided-Dice-Simulator project files contains two information files of sixsided dice. These information files have already been imported into the corresponding working folder of the Sixsided Dice Simulator, and you can use them simply by launching the Sixsided Dice Simulator as usual.

You can also create your own sixsided dice by referring to the sixsided dice information file provided in the project files and import them into the Sixsided Dice Simulator.
